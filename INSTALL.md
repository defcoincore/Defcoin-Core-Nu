Building Defcoin
================

See doc/build-*.md for instructions on building the various
elements of the Defcoin Core reference implementation of Defcoin.

For current public build guidance, start with a fresh clone and read the
canonical Defcoin Core Nu technical guide:

```sh
git clone https://github.com/DefcoinCore/Defcoin-Core-Nu.git
cd Defcoin-Core-Nu
```

See `doc/defcoin-core-nu-technical-guide.md` for Apple Silicon, Intel macOS,
and Windows build notes. Keep workstation-specific paths and local machine
details out of committed documentation.
