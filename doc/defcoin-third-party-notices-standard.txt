Defcoin Core Standard Build - License Notices
=============================================

This standard build contains the Defcoin Core codebase derived from Litecoin
Core and Bitcoin Core, plus the dependency libraries linked by the build system.

Primary linked libraries and bundled code
-----------------------------------------

The standard build uses the same dependency family as the base Litecoin Core
desktop wallet:

- Qt 5: LGPLv3 / GPLv3 / commercial licensing; linked as the Qt GUI toolkit.
- Berkeley DB 4.8: Oracle Berkeley DB license; used for legacy wallet storage.
- SQLite: public domain; used for descriptor wallet database support.
- Boost: Boost Software License 1.0; used throughout the C++ codebase.
- OpenSSL/libcrypto: Apache License 2.0; used for cryptographic support from
  the linked system/dependency library.
- libevent: BSD-style license; used for network event handling.
- LevelDB: BSD-style license; used for block index and chainstate storage.
- secp256k1 and secp256k1-zkp code: MIT license; used for elliptic-curve
  cryptography and inherited Litecoin MWEB-related code paths.
- CRC32C: BSD-style license; used by LevelDB-related storage code.
- Unbound: BSD-style license when built in by the dependency environment; used
  for DNS resolution support where enabled.
- QRencode: LGPLv2.1+; used by the GUI to render payment request QR codes.
- zlib/libpng and related image libraries: zlib/libpng-style licenses; used by
  Qt and the GUI image pipeline where linked by the build environment.

Standard build assets
---------------------

The standard build uses Defcoin branding assets and the base wallet icon set
needed for the normal wallet and node controls. Defcoin-specific icon artwork
in this tree is provided for this project and is distributed with the Defcoin
Core build. Upstream Bitcoin/Litecoin-origin icons and UI assets retain their
upstream MIT-compatible licensing.

DC34 Edition-only assets and features are intentionally not listed here because
they are not part of the standard package.
